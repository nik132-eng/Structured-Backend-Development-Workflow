# Status

Report where the current task stands, briefly: what's done (with evidence), what's in progress, what's left from the plan/progress log, and any blockers. If a task folder exists under `docs/agent-tasks/`, read `02-implementation-plan.md` and `03-progress-log.md` for the source of truth. No code changes.
